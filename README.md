# Portfolio (static)

This repository contains a simple static portfolio site (HTML + CSS). Deploy to GitHub Pages by serving the repository's `gh-pages` branch (or using the `docs/` folder or GitHub Pages settings).

How to deploy locally (recommended quick steps):

1. Clone the repo
   - git clone git@github.com:USERNAME/REPO.git
2. Create and switch to gh-pages branch
   - git checkout -b gh-pages
3. Add files (index.html, assets/style.css, README.md)
4. Commit and push
   - git add .
   - git commit -m "Add static portfolio site"
   - git push -u origin gh-pages
